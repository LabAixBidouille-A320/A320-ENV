# FMGS par JeeHells

## Introduction

La suite FMGS est un ensemble de plusieurs modules permettant de
s'interfacer avec les logiciels Prepar3D et/ou Flight Simulator X. Cette
suite logicielle permet également l'affichage des données de simulation
sur des écrans typés avioniques, comme le PFD, le ND... Ces derniers
permettent alors au pilote d'avoir ses instruments sous les yeux, qui
peuvent être affichés comme parts d'un simulateur si le matériel associé
est adéquat.

FMGS permet également la connexion de plusieurs périphériques externes
directement dans le logiciel de simulation (P3D ou FSX). Il est alors
plus commode à l'utilisation d'opérer sur les appareils physiques.

Le développeur JeeHells laisse une porte d'accès à son logiciel via
le module HardwareConnect : il est possible de créer des bibliothèques
qui vont directement interfacer avec FMGS, et en conséquent, avec le
simulateur.
